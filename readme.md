Requirement: Node

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production
$ npm run build
```

## Deploy a new version

To deploy a new version of the site push your changes to the ```main``` branch.
